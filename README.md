## Installation:
1. Clone the repository:  
`git clone https://github.com/RohitShah1706/SE-Arithmetic-Operations.git`

2. Change directory:  
`cd SE-Arithmetic-Operations`

3. Compile the program:
- For Linux: 
```bash
    chmod +x install.sh
    ./install.sh
```
- For Windows:  
```bash
    ./install.ps1
```

4. Run the program:  
`./main.out` for Linux  
`./main.exe` for Windows
